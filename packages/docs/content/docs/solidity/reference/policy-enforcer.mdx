## State Variables

### MAX_PATH_DEPTH

Path scratch capacity in steps (be16 per step). Must equal `PF.MAX_PATH_DEPTH`;
a local literal is required because cross-library constants cannot size fixed arrays.

[Git Source](https://github.com/provencraft/callcium/blob/main/packages/contracts/src/PolicyEnforcer.sol#L76)

```solidity
uint8 internal constant MAX_PATH_DEPTH = 32
```

## Functions

### enforce

Reverts when `callData` violates `policy`.

[Git Source](https://github.com/provencraft/callcium/blob/main/packages/contracts/src/PolicyEnforcer.sol#L110)

```solidity
function enforce(bytes memory policy, bytes calldata callData) internal view;
```

**Parameters**

| Name       | Type    | Description                               |
| ---------- | ------- | ----------------------------------------- |
| `policy`   | `bytes` | The policy blob with embedded descriptor. |
| `callData` | `bytes` | The calldata to validate.                 |

### check

Returns true if `callData` complies with `policy`.

Reverts for malformed policies; returns false for violations.

[Git Source](https://github.com/provencraft/callcium/blob/main/packages/contracts/src/PolicyEnforcer.sol#L120)

```solidity
function check(bytes memory policy, bytes calldata callData) internal view returns (bool ok);
```

**Parameters**

| Name       | Type    | Description                               |
| ---------- | ------- | ----------------------------------------- |
| `policy`   | `bytes` | The policy blob with embedded descriptor. |
| `callData` | `bytes` | The calldata to validate.                 |

**Returns**

| Name | Type   | Description                                |
| ---- | ------ | ------------------------------------------ |
| `ok` | `bool` | True if calldata complies with the policy. |
